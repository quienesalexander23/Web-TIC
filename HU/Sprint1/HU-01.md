# HU-01: Autenticación Segura con JWT y 2FA

## 1. Descripción
- **Como** Usuario Final
- **Quiero** iniciar sesión en el sistema mediante mis credenciales y un código de doble factor (2FA)
- **Para** para garantizar que solo yo pueda acceder a mis recursos y mantener la seguridad institucional

## 2. Criterios de Aceptación
1. **Dado que** ingreso credenciales válidas, **cuando** el sistema las procesa, **entonces** me envía un código OTP de 6 dígitos a mi correo electrónico.
2. **Dado que** recibo el código OTP, **cuando** lo ingreso correctamente en menos de 5 minutos, **entonces** se me otorga un token JWT y accedo al sistema.
3. **Dado que** soy un atacante, **cuando** fallo el inicio de sesión 5 veces consecutivas, **entonces** el sistema bloquea temporalmente la cuenta (Lockout) por motivos de fuerza bruta.

## 3. Tareas Técnicas
- [ ] Implementar endpoint `POST /api/auth/login`.
- [ ] Configurar servicio de envío de correos SMTP.
- [ ] Implementar endpoint `POST /api/auth/verify-2fa` y generar token JWT.
- [ ] Configurar política de bloqueo por fuerza bruta en ASP.NET Core Identity.

## 4. Notas Adicionales
