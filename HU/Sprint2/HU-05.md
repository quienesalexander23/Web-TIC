# HU-05: Control de Acceso Basado en Roles (RBAC)

## 1. Descripción
- **Como** Administrador de Seguridad
- **Quiero** restringir el acceso a los endpoints y vistas del sistema dependiendo del rol del usuario autenticado
- **Para** para garantizar que un Docente o Estudiante no pueda consumir recursos administrativos, previniendo la escalada de privilegios

## 2. Criterios de Aceptación
1. **Dado que** soy un Docente, **cuando** intento acceder al panel de administración, **entonces** el backend rechaza la petición con HTTP 403 Forbidden.
2. **Dado que** soy un Docente, **cuando** navego en el frontend, **entonces** las opciones del menú exclusivas para Administradores se ocultan dinámicamente.

## 3. Tareas Técnicas
- [ ] Decorar endpoints críticos con `[Authorize(Roles='Administrador')]`.
- [ ] Extraer el claim de rol del token JWT en Angular.
- [ ] Crear la directiva `hasRole` en Angular para ocultar elementos del DOM.

## 4. Notas Adicionales
- Revisar que la implementación cumpla con las normativas OWASP Top 10 aplicables para esta sección.
