# HU-06: Asignación Dinámica de Permisos a Roles

## 1. Descripción
- **Como** Administrador
- **Quiero** poder gestionar y asignar permisos granulares a los roles existentes
- **Para** para flexibilizar el modelo de seguridad sin tener que recompilar la aplicación

## 2. Criterios de Aceptación
1. **Dado que** soy administrador, **cuando** accedo a la gestión de roles, **entonces** puedo ver los permisos atados a cada rol.
2. **Dado que** modifico los permisos de un rol, **cuando** guardo los cambios, **entonces** la matriz de accesos se actualiza en la base de datos.

## 3. Tareas Técnicas
- [ ] Crear modelo de datos relacional para Roles y Permisos.
- [ ] Implementar endpoints en `RolePermissionsController`.
- [ ] Construir interfaz en Angular para asignación visual de permisos mediante checkboxes.

## 4. Notas Adicionales
- Revisar que la implementación cumpla con las normativas OWASP Top 10 aplicables para esta sección.
