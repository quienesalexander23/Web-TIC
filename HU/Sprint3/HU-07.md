# HU-07: Registro Automático de Auditoría

## 1. Descripción
- **Como** Oficial de Seguridad
- **Quiero** que el sistema registre automáticamente quién, cuándo y qué acción administrativa se ejecutó
- **Para** para mantener una trazabilidad inmutable ante incidentes y cumplir con las normativas institucionales

## 2. Criterios de Aceptación
1. **Dado que** un Administrador edita un usuario, **cuando** se guarda el cambio, **entonces** se genera un registro en la tabla de Logs con el ID del administrador, la acción y la fecha.
2. **Dado que** ocurre una excepción no controlada en el backend, **cuando** el middleware la intercepta, **entonces** el error detallado se guarda en el log.

## 3. Tareas Técnicas
- [ ] Implementar un middleware global de captura de logs y excepciones.
- [ ] Crear entidad y tabla de Auditoría en Entity Framework Core.
- [ ] Crear endpoint de solo-lectura para consultar los logs generados.

## 4. Notas Adicionales
- Revisar que la implementación cumpla con las normativas OWASP Top 10 aplicables para esta sección.
