# HU-08: Dashboard de Métricas del Sistema

## 1. Descripción
- **Como** Administrador
- **Quiero** visualizar un resumen estadístico de la distribución de usuarios y sus estados de conexión
- **Para** para monitorear rápidamente el uso general del módulo y la salud del sistema

## 2. Criterios de Aceptación
1. **Dado que** accedo a la página principal del panel, **cuando** la vista carga, **entonces** se renderizan tarjetas con KPIs (total usuarios, bloqueados, inactivos).
2. **Dado que** los datos cambian en la base de datos, **cuando** refresco la vista, **entonces** las gráficas y métricas se actualizan de manera performante.

## 3. Tareas Técnicas
- [ ] Crear endpoint `/api/stats/dashboard` en el backend para agrupar métricas.
- [ ] Diseñar el Dashboard en Angular usando componentes de tarjetas visuales.
- [ ] Asegurar que la consulta estadística sea optimizada en EF Core.

## 4. Notas Adicionales
- Revisar que la implementación cumpla con las normativas OWASP Top 10 aplicables para esta sección.
